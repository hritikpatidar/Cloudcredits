# Responsive Landing Page

A modern and responsive Landing Page built using React.js and Tailwind CSS.

## Features

* Responsive Navigation Bar
* Hero Section
* Features Section
* About Section
* Contact Section
* Footer Section
* Mobile Friendly Design
* Modern UI/UX

## Tech Stack

* React.js
* Tailwind CSS
* JavaScript
* Vite

## Installation

```bash
npm install
npm run dev
```

## Project Structure

```bash
src/
├── components/
├── App.jsx
├── main.jsx
└── index.css
```

## Learning Outcomes

* React Component Architecture
* Responsive Web Design
* Tailwind CSS Styling
* UI/UX Development

* Result Screen
